<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with JoltC in minutes
  </p>
  
  <Steps>
    <Step title="Install JoltC">
      Build JoltC from source using CMake. The library supports both static and shared builds.
      
      ```bash
      git clone https://github.com/amerkoleci/joltc.git
      cd joltc
      mkdir build && cd build
      cmake ..
      cmake --build .
      ```
    </Step>
    
    <Step title="Initialize the Physics System">
      Set up the physics system with your collision layers and job system.
      
      ```c
      #include "joltc.h"

      // Initialize Jolt
      JPH_Init();

      // Create job system for multi-threading
      JPH_JobSystem* jobSystem = JPH_JobSystemThreadPool_Create(NULL);

      // Set up collision filtering
      JPH_ObjectLayerPairFilter* objectFilter = JPH_ObjectLayerPairFilterTable_Create(2);
      JPH_BroadPhaseLayerInterface* broadPhaseInterface = JPH_BroadPhaseLayerInterfaceTable_Create(2, 2);
      
      // Create physics system
      JPH_PhysicsSystemSettings settings = {
          .maxBodies = 65536,
          .broadPhaseLayerInterface = broadPhaseInterface,
          .objectLayerPairFilter = objectFilter
      };
      JPH_PhysicsSystem* system = JPH_PhysicsSystem_Create(&settings);
      ```
    </Step>
    
    <Step title="Create Your First Body">
      Add a dynamic sphere that responds to gravity.
      
      ```c
      // Get the body interface
      JPH_BodyInterface* bodyInterface = JPH_PhysicsSystem_GetBodyInterface(system);

      // Create a sphere shape
      JPH_SphereShape* sphere = JPH_SphereShape_Create(0.5f);

      // Set initial position
      JPH_Vec3 position = { 0.0f, 10.0f, 0.0f };

      // Create body settings
      JPH_BodyCreationSettings* bodySettings = JPH_BodyCreationSettings_Create3(
          (JPH_Shape*)sphere,
          &position,
          NULL, // Identity rotation
          JPH_MotionType_Dynamic,
          LAYER_MOVING
      );

      // Add body to the world
      JPH_BodyID bodyId = JPH_BodyInterface_CreateAndAddBody(
          bodyInterface, 
          bodySettings, 
          JPH_Activation_Activate
      );
      
      JPH_BodyCreationSettings_Destroy(bodySettings);
      ```
      
      <Note>
        The body is now part of your physics simulation and will fall under gravity!
      </Note>
    </Step>
    
    <Step title="Run the Simulation">
      Update the physics world each frame.
      
      ```c
      const float deltaTime = 1.0f / 60.0f; // 60 FPS
      const int collisionSteps = 1;

      // Simulate one step
      JPH_PhysicsSystem_Update(system, deltaTime, collisionSteps, jobSystem);

      // Get the updated position
      JPH_RVec3 newPosition;
      JPH_BodyInterface_GetPosition(bodyInterface, bodyId, &newPosition);
      printf("Position: (%f, %f, %f)\n", newPosition.x, newPosition.y, newPosition.z);
      ```
      
      <Tip>
        For more complex simulations, check out the [Physics System guide](/core/physics-system).
      </Tip>
    </Step>
  </Steps>
</div>
