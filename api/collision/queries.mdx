## Overview

JoltC provides broad-phase and narrow-phase collision query interfaces for detecting collisions, performing raycasts, and shape casts in the physics world.

## Broad Phase Query

The broad phase performs fast, coarse collision detection using bounding boxes.

### Get Broad Phase Query

<CodeGroup>
```c JPH_PhysicsSystem_GetBroadPhaseQuery
const JPH_BroadPhaseQuery* JPH_PhysicsSystem_GetBroadPhaseQuery(
    const JPH_PhysicsSystem* system
);
```
</CodeGroup>

Retrieve the broad phase query interface from the physics system.

<ParamField path="system" type="const JPH_PhysicsSystem*">
  The physics system instance
</ParamField>

<ResponseField name="return" type="const JPH_BroadPhaseQuery*">
  Pointer to the broad phase query interface
</ResponseField>

### Raycast (Broad Phase)

<CodeGroup>
```c JPH_BroadPhaseQuery_CastRay
bool JPH_BroadPhaseQuery_CastRay(
    const JPH_BroadPhaseQuery* query,
    const JPH_Vec3* origin,
    const JPH_Vec3* direction,
    JPH_RayCastBodyCollectorCallback* callback,
    void* userData,
    const JPH_BroadPhaseLayerFilter* broadPhaseLayerFilter,
    const JPH_ObjectLayerFilter* objectLayerFilter
);
```
</CodeGroup>

Cast a ray through the broad phase and collect all potentially intersecting bodies.

<ParamField path="query" type="const JPH_BroadPhaseQuery*">
  The broad phase query interface
</ParamField>

<ParamField path="origin" type="const JPH_Vec3*">
  Ray origin point
</ParamField>

<ParamField path="direction" type="const JPH_Vec3*">
  Ray direction (length determines max distance)
</ParamField>

<ParamField path="callback" type="JPH_RayCastBodyCollectorCallback*">
  Callback function to receive results: `float callback(void* userData, const JPH_BroadPhaseCastResult* result)`
  
  Return the new early out fraction (1.0 to continue with all results)
</ParamField>

<ParamField path="userData" type="void*">
  User data passed to callback
</ParamField>

<ParamField path="broadPhaseLayerFilter" type="const JPH_BroadPhaseLayerFilter*">
  Optional filter for broad phase layers (can be NULL)
</ParamField>

<ParamField path="objectLayerFilter" type="const JPH_ObjectLayerFilter*">
  Optional filter for object layers (can be NULL)
</ParamField>

Cast a ray with a specific collector type (AllHit, ClosestHit, etc.).

<ParamField path="collectorType" type="JPH_CollisionCollectorType">
  Type of collector:
  - `JPH_CollisionCollectorType_AllHit` - Collect all hits
  - `JPH_CollisionCollectorType_AllHitSorted` - Collect all hits sorted by distance
  - `JPH_CollisionCollectorType_ClosestHit` - Only the closest hit
  - `JPH_CollisionCollectorType_AnyHit` - Any hit (fastest)
</ParamField>

<ParamField path="callback" type="JPH_RayCastBodyResultCallback*">
  Result callback: `void callback(void* userData, const JPH_BroadPhaseCastResult* result)`
</ParamField>

### Raycast (Narrow Phase)

<CodeGroup>
```c JPH_NarrowPhaseQuery_CastRay
bool JPH_NarrowPhaseQuery_CastRay(
    const JPH_NarrowPhaseQuery* query,
    const JPH_RVec3* origin,
    const JPH_Vec3* direction,
    JPH_RayCastResult* hit,
    const JPH_BroadPhaseLayerFilter* broadPhaseLayerFilter,
    const JPH_ObjectLayerFilter* objectLayerFilter,
    const JPH_BodyFilter* bodyFilter
);
```
</CodeGroup>

Cast a ray and get the closest hit.

<ParamField path="origin" type="const JPH_RVec3*">
  Ray origin (uses real precision for position)
</ParamField>

<ParamField path="direction" type="const JPH_Vec3*">
  Ray direction (length determines max distance)
</ParamField>

<ParamField path="hit" type="JPH_RayCastResult*">
  Output result structure containing:
  - `bodyID` - The body that was hit
  - `fraction` - Fraction along ray where hit occurred (0-1)
  - `subShapeID2` - Sub-shape that was hit
</ParamField>

<ParamField path="bodyFilter" type="const JPH_BodyFilter*">
  Optional filter for bodies (can be NULL)
</ParamField>

<ResponseField name="return" type="bool">
  True if a hit was found
</ResponseField>

## Example Usage

<CodeGroup>
```c Raycast Example
// Simple raycast
JPH_RayCastResult hit;
JPH_Vec3 origin = {0, 10, 0};
JPH_Vec3 direction = {0, -20, 0}; // Ray goes down 20 units

const JPH_NarrowPhaseQuery* query = 
    JPH_PhysicsSystem_GetNarrowPhaseQuery(physics_system);

if (JPH_NarrowPhaseQuery_CastRay(query, &origin, &direction, &hit,
    NULL, NULL, NULL)) {
    printf("Hit body %u at fraction %f\n", hit.bodyID, hit.fraction);
}
```

```c Shape Cast Example
// Cast a sphere forward
JPH_SphereShape* sphere = JPH_SphereShape_Create(1.0f);

JPH_RMat4 transform;
JPH_RMat4_Identity(&transform);
transform.column3.x = 0;
transform.column3.y = 5;
transform.column3.z = 0;

JPH_Vec3 direction = {0, 0, 10}; // Cast 10 units forward

JPH_ShapeCastSettings settings;
JPH_ShapeCastSettings_Init(&settings);

JPH_ShapeCastResult result;
float closest_fraction = 1.0f;

float shape_cast_callback(void* ctx, const JPH_ShapeCastResult* res) {
    if (res->fraction < closest_fraction) {
        closest_fraction = res->fraction;
        memcpy(&result, res, sizeof(JPH_ShapeCastResult));
    }
    return closest_fraction; // Early out
}

const JPH_NarrowPhaseQuery* query = 
    JPH_PhysicsSystem_GetNarrowPhaseQuery(physics_system);

JPH_RVec3 baseOffset = {0, 0, 0};
JPH_NarrowPhaseQuery_CastShape(query, (JPH_Shape*)sphere, &transform,
    &direction, &settings, &baseOffset, shape_cast_callback, NULL,
    NULL, NULL, NULL, NULL);

JPH_Shape_Destroy((JPH_Shape*)sphere);
```

```c Point Test Example
// Test if a point is inside any body
JPH_RVec3 test_point = {5.0, 2.0, 3.0};

typedef struct {
    bool found;
    JPH_BodyID bodyID;
} PointTestContext;

float point_callback(void* ctx, const JPH_CollidePointResult* result) {
    PointTestContext* context = (PointTestContext*)ctx;
    context->found = true;
    context->bodyID = result->bodyID;
    return 0.0f; // Stop on first hit
}

PointTestContext context = {false, 0};
const JPH_NarrowPhaseQuery* query = 
    JPH_PhysicsSystem_GetNarrowPhaseQuery(physics_system);

JPH_NarrowPhaseQuery_CollidePoint(query, &test_point, point_callback,
    &context, NULL, NULL, NULL, NULL);

if (context.found) {
    printf("Point is inside body %u\n", context.bodyID);
}
```
</CodeGroup>
