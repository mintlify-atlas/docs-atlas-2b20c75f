---
title: Body Interface
description: Functions for managing rigid bodies in the physics world
---

The `JPH_BodyInterface` provides the main API for creating, adding, removing, and manipulating rigid bodies in the physics simulation. This is the primary interface for body operations.

## Body Creation

### JPH_BodyInterface_CreateBody

Creates a body but does not add it to the physics system.

<ParamField path="bodyInterface" type="JPH_BodyInterface*">
  The body interface
</ParamField>

<ParamField path="settings" type="const JPH_BodyCreationSettings*">
  Body creation settings
</ParamField>

**Returns:** `JPH_Body*` - The created body

### JPH_BodyInterface_CreateBodyWithID

Creates a body with a specific ID.

<ParamField path="bodyInterface" type="JPH_BodyInterface*">
  The body interface
</ParamField>

<ParamField path="bodyID" type="JPH_BodyID">
  The desired body ID
</ParamField>

<ParamField path="settings" type="const JPH_BodyCreationSettings*">
  Body creation settings
</ParamField>

**Returns:** `JPH_Body*` - The created body

### JPH_BodyInterface_CreateBodyWithoutID

Creates a body without assigning it an ID.

<ParamField path="bodyInterface" type="JPH_BodyInterface*">
  The body interface
</ParamField>

<ParamField path="settings" type="const JPH_BodyCreationSettings*">
  Body creation settings
</ParamField>

**Returns:** `JPH_Body*` - The created body

### JPH_BodyInterface_CreateAndAddBody

Creates a body and adds it to the physics system in one operation.

<ParamField path="bodyInterface" type="JPH_BodyInterface*">
  The body interface
</ParamField>

<ParamField path="settings" type="const JPH_BodyCreationSettings*">
  Body creation settings
</ParamField>

<ParamField path="activationMode" type="JPH_Activation">
  Whether to activate the body immediately (`JPH_Activation_Activate` or `JPH_Activation_DontActivate`)
</ParamField>

**Returns:** `JPH_BodyID` - The ID of the created body

```c
JPH_BodyCreationSettings* settings = JPH_BodyCreationSettings_Create3(
    shape, &position, &rotation, JPH_MotionType_Dynamic, objectLayer);

JPH_BodyID bodyID = JPH_BodyInterface_CreateAndAddBody(
    bodyInterface, settings, JPH_Activation_Activate);

JPH_BodyCreationSettings_Destroy(settings);
```
