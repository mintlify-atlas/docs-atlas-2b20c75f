# Hinge Constraint

The `JPH_HingeConstraint` creates a hinge joint that allows rotation around a single axis while constraining all other degrees of freedom. It supports angular limits, motors, and friction.

## Use Cases

- Doors and gates
- Wheels and axles
- Robotic joints
- Rotating platforms
- Powered hinges with motors
- Limited rotation joints (e.g., laptop screen)

## Types

### JPH_HingeConstraint

```c
typedef struct JPH_HingeConstraint JPH_HingeConstraint;
```

Opaque handle to a hinge constraint instance.

### JPH_HingeConstraintSettings

```c
typedef struct JPH_HingeConstraintSettings {
    JPH_ConstraintSettings  base;
    JPH_ConstraintSpace     space;
    JPH_RVec3               point1;
    JPH_Vec3                hingeAxis1;
    JPH_Vec3                normalAxis1;
    JPH_RVec3               point2;
    JPH_Vec3                hingeAxis2;
    JPH_Vec3                normalAxis2;
    float                   limitsMin;
    float                   limitsMax;
    JPH_SpringSettings      limitsSpringSettings;
    float                   maxFrictionTorque;
    JPH_MotorSettings       motorSettings;
} JPH_HingeConstraintSettings;
```

Settings structure for creating a hinge constraint.

<ParamField path="base" type="JPH_ConstraintSettings">
  Base constraint settings (enabled, priority, etc.)
</ParamField>

<ParamField path="space" type="JPH_ConstraintSpace">
  Coordinate space for the constraint points and axes
</ParamField>

<ParamField path="point1" type="JPH_RVec3">
  Hinge attachment point on body 1
</ParamField>

<ParamField path="hingeAxis1" type="JPH_Vec3">
  Rotation axis direction for body 1 (normalized)
</ParamField>

<ParamField path="normalAxis1" type="JPH_Vec3">
  Reference axis perpendicular to hinge axis for body 1 (for measuring angle)
</ParamField>

<ParamField path="point2" type="JPH_RVec3">
  Hinge attachment point on body 2
</ParamField>

<ParamField path="hingeAxis2" type="JPH_Vec3">
  Rotation axis direction for body 2 (normalized)
</ParamField>

<ParamField path="normalAxis2" type="JPH_Vec3">
  Reference axis perpendicular to hinge axis for body 2
</ParamField>

<ParamField path="limitsMin" type="float">
  Minimum rotation angle in radians (use -π to +π range)
</ParamField>

<ParamField path="limitsMax" type="float">
  Maximum rotation angle in radians
</ParamField>

<ParamField path="limitsSpringSettings" type="JPH_SpringSettings">
  Spring settings for angular limits (soft limits)
</ParamField>

<ParamField path="maxFrictionTorque" type="float">
  Maximum friction torque applied to the hinge (N·m)
</ParamField>

<ParamField path="motorSettings" type="JPH_MotorSettings">
  Motor settings for powered rotation
</ParamField>

## Example

```c
// Create a door hinge
JPH_Body* doorFrame = /* ... static body ... */;
JPH_Body* door = /* ... dynamic body ... */;

// Initialize constraint settings
JPH_HingeConstraintSettings settings;
JPH_HingeConstraintSettings_Init(&settings);

// Configure hinge
settings.space = JPH_ConstraintSpace_WorldSpace;
settings.point1 = (JPH_RVec3){ 0, 1, 0 };  // Hinge at door edge
settings.point2 = (JPH_RVec3){ 0, 1, 0 };
settings.hingeAxis1 = (JPH_Vec3){ 0, 1, 0 };  // Y-axis rotation
settings.hingeAxis2 = (JPH_Vec3){ 0, 1, 0 };
settings.normalAxis1 = (JPH_Vec3){ 1, 0, 0 };
settings.normalAxis2 = (JPH_Vec3){ 1, 0, 0 };

// Set limits: door can swing 0 to 120 degrees
settings.limitsMin = 0.0f;
settings.limitsMax = 2.09f;  // ~120 degrees in radians

// Add friction for realistic feel
settings.maxFrictionTorque = 5.0f;

// Create the constraint
JPH_HingeConstraint* hinge = JPH_HingeConstraint_Create(
    &settings,
    doorFrame,
    door
);

JPH_PhysicsSystem_AddConstraint(physicsSystem, (JPH_Constraint*)hinge);

// Later: Add motor to automatically close door
JPH_HingeConstraint_SetMotorState(hinge, JPH_MotorState_Position);
JPH_HingeConstraint_SetTargetAngle(hinge, 0.0f);  // Target: closed
```
