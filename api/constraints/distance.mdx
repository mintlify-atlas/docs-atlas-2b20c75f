# Distance Constraint

The `JPH_DistanceConstraint` maintains a specific distance range between two points attached to two bodies. It allows rotation but restricts the distance between the attachment points to remain within minimum and maximum bounds.

## Use Cases

- Rope or chain simulation
- Cable connections
- Elastic connections between objects
- Minimum/maximum distance enforcement
- Spring-like behavior with spring settings

## Types

### JPH_DistanceConstraint

```c
typedef struct JPH_DistanceConstraint JPH_DistanceConstraint;
```

Opaque handle to a distance constraint instance.

### JPH_DistanceConstraintSettings

```c
typedef struct JPH_DistanceConstraintSettings {
    JPH_ConstraintSettings  base;
    JPH_ConstraintSpace     space;
    JPH_RVec3               point1;
    JPH_RVec3               point2;
    float                   minDistance;
    float                   maxDistance;
    JPH_SpringSettings      limitsSpringSettings;
} JPH_DistanceConstraintSettings;
```

Settings structure for creating a distance constraint.

<ParamField path="base" type="JPH_ConstraintSettings">
  Base constraint settings (enabled, priority, etc.)
</ParamField>

<ParamField path="space" type="JPH_ConstraintSpace">
  Coordinate space for the constraint points:
  - `JPH_ConstraintSpace_LocalToBodyCOM`: Local to each body's center of mass
  - `JPH_ConstraintSpace_WorldSpace`: World space coordinates
</ParamField>

<ParamField path="point1" type="JPH_RVec3">
  Attachment point on body 1
</ParamField>

<ParamField path="point2" type="JPH_RVec3">
  Attachment point on body 2
</ParamField>

<ParamField path="minDistance" type="float">
  Minimum allowed distance between the points (meters)
</ParamField>

<ParamField path="maxDistance" type="float">
  Maximum allowed distance between the points (meters)
</ParamField>

<ParamField path="limitsSpringSettings" type="JPH_SpringSettings">
  Spring settings applied when limits are violated. Use this to make the constraint soft/elastic.
</ParamField>

## Example

```c
// Create two bodies
JPH_Body* ball1 = /* ... create first ball ... */;
JPH_Body* ball2 = /* ... create second ball ... */;

// Initialize constraint settings
JPH_DistanceConstraintSettings settings;
JPH_DistanceConstraintSettings_Init(&settings);

// Configure as a rope with 5 meter length
settings.space = JPH_ConstraintSpace_WorldSpace;
settings.point1 = (JPH_RVec3){ 0, 5, 0 };
settings.point2 = (JPH_RVec3){ 0, 0, 0 };
settings.minDistance = 0.0f;    // Can be compressed
settings.maxDistance = 5.0f;    // 5 meter maximum length

// Make it stretchy with spring settings
settings.limitsSpringSettings.mode = JPH_SpringMode_FrequencyAndDamping;
settings.limitsSpringSettings.frequencyOrStiffness = 2.0f;
settings.limitsSpringSettings.damping = 0.5f;

// Create the constraint
JPH_DistanceConstraint* constraint = JPH_DistanceConstraint_Create(
    &settings,
    ball1,
    ball2
);

// Add to physics system
JPH_PhysicsSystem_AddConstraint(physicsSystem, (JPH_Constraint*)constraint);

// Later: Dynamically adjust the rope length
JPH_DistanceConstraint_SetDistance(constraint, 0.0f, 3.0f);
```
