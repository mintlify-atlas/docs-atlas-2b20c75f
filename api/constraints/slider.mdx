# Slider Constraint

The `JPH_SliderConstraint` creates a sliding joint that allows linear movement along a single axis while constraining all other degrees of freedom. It supports position limits, motors, and friction.

## Use Cases

- Pistons and hydraulic cylinders
- Drawer slides
- Linear actuators
- Elevator mechanisms
- Spring-loaded sliding components
- Rail-based movement

## Types

### JPH_SliderConstraint

```c
typedef struct JPH_SliderConstraint JPH_SliderConstraint;
```

Opaque handle to a slider constraint instance.

### JPH_SliderConstraintSettings

```c
typedef struct JPH_SliderConstraintSettings {
    JPH_ConstraintSettings  base;
    JPH_ConstraintSpace     space;
    bool                    autoDetectPoint;
    JPH_RVec3               point1;
    JPH_Vec3                sliderAxis1;
    JPH_Vec3                normalAxis1;
    JPH_RVec3               point2;
    JPH_Vec3                sliderAxis2;
    JPH_Vec3                normalAxis2;
    float                   limitsMin;
    float                   limitsMax;
    JPH_SpringSettings      limitsSpringSettings;
    float                   maxFrictionForce;
    JPH_MotorSettings       motorSettings;
} JPH_SliderConstraintSettings;
```

Settings structure for creating a slider constraint.

<ParamField path="base" type="JPH_ConstraintSettings">
  Base constraint settings (enabled, priority, etc.)
</ParamField>

<ParamField path="space" type="JPH_ConstraintSpace">
  Coordinate space for the constraint points and axes
</ParamField>

<ParamField path="autoDetectPoint" type="bool">
  If true, automatically calculate the constraint point from body positions
</ParamField>

<ParamField path="point1" type="JPH_RVec3">
  Slider attachment point on body 1
</ParamField>

<ParamField path="sliderAxis1" type="JPH_Vec3">
  Direction of allowed sliding motion for body 1 (normalized)
</ParamField>

<ParamField path="normalAxis1" type="JPH_Vec3">
  Reference axis perpendicular to slider axis for body 1
</ParamField>

<ParamField path="point2" type="JPH_RVec3">
  Slider attachment point on body 2
</ParamField>

<ParamField path="sliderAxis2" type="JPH_Vec3">
  Direction of allowed sliding motion for body 2 (normalized)
</ParamField>

<ParamField path="normalAxis2" type="JPH_Vec3">
  Reference axis perpendicular to slider axis for body 2
</ParamField>

<ParamField path="limitsMin" type="float">
  Minimum position along slider axis (meters)
</ParamField>

<ParamField path="limitsMax" type="float">
  Maximum position along slider axis (meters)
</ParamField>

<ParamField path="limitsSpringSettings" type="JPH_SpringSettings">
  Spring settings for position limits (soft limits)
</ParamField>

<ParamField path="maxFrictionForce" type="float">
  Maximum friction force applied to the slider (N)
</ParamField>

<ParamField path="motorSettings" type="JPH_MotorSettings">
  Motor settings for powered sliding
</ParamField>

## Example

```c
// Create a piston mechanism
JPH_Body* cylinder = /* ... static body ... */;
JPH_Body* piston = /* ... dynamic body ... */;

// Initialize constraint settings
JPH_SliderConstraintSettings settings;
JPH_SliderConstraintSettings_Init(&settings);

// Configure slider
settings.space = JPH_ConstraintSpace_WorldSpace;
settings.autoDetectPoint = true;

JPH_Vec3 sliderAxis = { 0, 1, 0 };  // Y-axis sliding
JPH_SliderConstraintSettings_SetSliderAxis(&settings, &sliderAxis);

// Set limits: piston can move 0.5 meters
settings.limitsMin = 0.0f;
settings.limitsMax = 0.5f;

// Add friction
settings.maxFrictionForce = 10.0f;

// Configure motor for powered movement
settings.motorSettings.springSettings.mode = JPH_SpringMode_FrequencyAndDamping;
settings.motorSettings.springSettings.frequencyOrStiffness = 5.0f;
settings.motorSettings.springSettings.damping = 1.0f;
settings.motorSettings.minForceLimit = -1000.0f;
settings.motorSettings.maxForceLimit = 1000.0f;

// Create the constraint
JPH_SliderConstraint* slider = JPH_SliderConstraint_Create(
    &settings,
    cylinder,
    piston
);

JPH_PhysicsSystem_AddConstraint(physicsSystem, (JPH_Constraint*)slider);

// Activate motor to extend piston
JPH_SliderConstraint_SetMotorState(slider, JPH_MotorState_Position);
JPH_SliderConstraint_SetTargetPosition(slider, 0.5f);  // Extend fully

// Later: Retract piston
JPH_SliderConstraint_SetTargetPosition(slider, 0.0f);
```
