# Six DOF Constraint

The `JPH_SixDOFConstraint` is the most flexible constraint type, providing independent control over all 6 degrees of freedom (3 translational and 3 rotational axes). Each axis can be configured as free, fixed, or limited.

## Use Cases

- Complex mechanical joints
- Character ragdoll joints
- Robotic arms and manipulators
- Custom constraints with specific DOF requirements
- Vehicle suspension with multiple controlled axes
- Joystick or flight control mechanisms

## Types


Opaque handle to a six DOF constraint instance.

## Example

```c
// Create a ragdoll shoulder joint
JPH_Body* torso = /* ... */;
JPH_Body* upperArm = /* ... */;

// Initialize settings
JPH_SixDOFConstraintSettings settings;
JPH_SixDOFConstraintSettings_Init(&settings);

// Set constraint space
settings.space = JPH_ConstraintSpace_WorldSpace;
settings.position1 = (JPH_RVec3){ 0.5, 1.5, 0 };  // Shoulder position
settings.position2 = (JPH_RVec3){ 0.5, 1.5, 0 };

// Configure axes
// Lock all translations
JPH_SixDOFConstraintSettings_MakeFixedAxis(&settings, JPH_SixDOFConstraintAxis_TranslationX);
JPH_SixDOFConstraintSettings_MakeFixedAxis(&settings, JPH_SixDOFConstraintAxis_TranslationY);
JPH_SixDOFConstraintSettings_MakeFixedAxis(&settings, JPH_SixDOFConstraintAxis_TranslationZ);

// Allow limited rotation around X and Y (swing)
JPH_SixDOFConstraintSettings_SetLimitedAxis(&settings,
    JPH_SixDOFConstraintAxis_RotationX, -1.57f, 1.57f);  // ±90 degrees
JPH_SixDOFConstraintSettings_SetLimitedAxis(&settings,
    JPH_SixDOFConstraintAxis_RotationY, -0.785f, 1.57f);  // -45 to +90 degrees

// Allow free rotation around Z (twist)
JPH_SixDOFConstraintSettings_SetLimitedAxis(&settings,
    JPH_SixDOFConstraintAxis_RotationZ, -3.14f, 3.14f);  // Full rotation

// Add some friction
for (int i = 0; i < 6; i++) {
    settings.maxFriction[i] = (i >= 3) ? 5.0f : 0.0f;  // Rotational friction only
}

// Create constraint
JPH_SixDOFConstraint* shoulder = JPH_SixDOFConstraint_Create(
    &settings,
    torso,
    upperArm
);

JPH_PhysicsSystem_AddConstraint(physicsSystem, (JPH_Constraint*)shoulder);
```
