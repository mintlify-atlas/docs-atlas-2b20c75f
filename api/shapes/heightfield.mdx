---
title: Height Field Shape
description: API reference for JoltC height field terrain shape
---

Height field shapes are used for efficient terrain representation. They store height samples in a regular grid and are optimized for large-scale terrain.

## Shape Creation

### JPH_HeightFieldShapeSettings_Create

Creates height field shape settings from height samples.

```c
JPH_HeightFieldShapeSettings* JPH_HeightFieldShapeSettings_Create(
    const float* samples,
    const JPH_Vec3* offset,
    const JPH_Vec3* scale,
    uint32_t sampleCount,
    const uint8_t* materialIndices
);
```

<ParamField path="samples" type="const float*">
  Array of height samples. Must be `sampleCount * sampleCount` elements in row-major order
</ParamField>

<ParamField path="offset" type="const JPH_Vec3*">
  World-space offset of the height field origin
</ParamField>

<ParamField path="scale" type="const JPH_Vec3*">
  Scale factor for each axis (x = horizontal spacing, y = height scale, z = horizontal spacing)
</ParamField>

<ParamField path="sampleCount" type="uint32_t">
  Number of samples along each axis (creates a sampleCount × sampleCount grid)
</ParamField>

<ParamField path="materialIndices" type="const uint8_t*">
  Optional array of material indices per sample. Can be `NULL` for default material
</ParamField>

**Returns:** Height field shape settings

**Example:**
```c
// Create a 256x256 height field
#define SAMPLE_COUNT 256
float* heights = malloc(SAMPLE_COUNT * SAMPLE_COUNT * sizeof(float));

// Fill with procedural terrain
for (uint32_t z = 0; z < SAMPLE_COUNT; z++) {
    for (uint32_t x = 0; x < SAMPLE_COUNT; x++) {
        float fx = (float)x / SAMPLE_COUNT;
        float fz = (float)z / SAMPLE_COUNT;
        heights[z * SAMPLE_COUNT + x] = sinf(fx * 10) * cosf(fz * 10) * 5.0f;
    }
}

JPH_Vec3 offset = {-128, 0, -128};
JPH_Vec3 scale = {1.0f, 1.0f, 1.0f};

JPH_HeightFieldShapeSettings* settings = JPH_HeightFieldShapeSettings_Create(
    heights,
    &offset,
    &scale,
    SAMPLE_COUNT,
    NULL  // No per-sample materials
);
```

## Complete Example

```c
// Create a 128x128 terrain with rolling hills
#define TERRAIN_SIZE 128
float* heights = malloc(TERRAIN_SIZE * TERRAIN_SIZE * sizeof(float));

for (uint32_t z = 0; z < TERRAIN_SIZE; z++) {
    for (uint32_t x = 0; x < TERRAIN_SIZE; x++) {
        float fx = (float)x / TERRAIN_SIZE * 6.28f;
        float fz = (float)z / TERRAIN_SIZE * 6.28f;
        heights[z * TERRAIN_SIZE + x] = sinf(fx) * cosf(fz) * 10.0f;
    }
}

// Position at origin, 1m grid spacing, scale heights by 1.0
JPH_Vec3 offset = {-TERRAIN_SIZE / 2.0f, 0, -TERRAIN_SIZE / 2.0f};
JPH_Vec3 scale = {1.0f, 1.0f, 1.0f};

JPH_HeightFieldShapeSettings* settings = JPH_HeightFieldShapeSettings_Create(
    heights,
    &offset,
    &scale,
    TERRAIN_SIZE,
    NULL
);

// Configure compression
uint32_t bitsPerSample = JPH_HeightFieldShapeSettings_CalculateBitsPerSampleForError(
    settings, 0.1f  // 10cm max error
);
JPH_HeightFieldShapeSettings_SetBitsPerSample(settings, bitsPerSample);

// Create the shape
JPH_HeightFieldShape* terrain = JPH_HeightFieldShapeSettings_CreateShape(settings);

free(heights);
```
