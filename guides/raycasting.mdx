# Raycasting and Collision Queries

JoltC provides powerful raycasting and collision detection tools for gameplay features like shooting, vision checks, and ground detection.

## Query Types

<CardGroup cols={2}>
  <Card title="Raycast" icon="lines-leaning">
    Cast an infinitely thin ray to find intersections
  </Card>
  <Card title="Shape Cast" icon="cube">
    Sweep a 3D shape along a direction
  </Card>
  <Card title="Collision Query" icon="circle-dot">
    Test if shapes overlap at a position
  </Card>
  <Card title="Point Query" icon="location-dot">

## Raycasting

### Basic Raycast

Cast a ray and get the first hit:

```c joltc.h:1971-1976
JPH_NarrowPhaseQuery* query = JPH_PhysicsSystem_GetNarrowPhaseQuery(system);

JPH_RVec3 origin = { 0.0f, 1.0f, 0.0f };
JPH_Vec3 direction = { 1.0f, 0.0f, 0.0f };
JPH_RayCastResult hit;

bool hasHit = JPH_NarrowPhaseQuery_CastRay(
    query,
    &origin,
    &direction,
    &hit,
    NULL,  // broadphase filter
    NULL,  // object layer filter
    NULL   // body filter
);

if (hasHit) {
    printf("Hit body %u at fraction %f\n", hit.bodyID, hit.fraction);
    
    // Calculate hit point
    JPH_RVec3 hitPoint;
    JPH_RRayCast_GetPointOnRay(&origin, &direction, hit.fraction, &hitPoint);
}
```

### Multiple Hits with Callback

Collect all hits along a ray:

<Tabs>
  <Tab title="Callback Function">
    ```c joltc.h:686-698
    typedef struct {
        int hitCount;
        JPH_BodyID hitBodies[32];
        float hitFractions[32];
    } RaycastContext;
    
    static float OnRayHit(void* context, const JPH_RayCastResult* result) {
        RaycastContext* ctx = (RaycastContext*)context;
        
        if (ctx->hitCount < 32) {
            ctx->hitBodies[ctx->hitCount] = result->bodyID;
            ctx->hitFractions[ctx->hitCount] = result->fraction;
            ctx->hitCount++;
        }
        
        // Return 1.0 to continue to end of ray
        // Return result->fraction to stop at this hit
        // Return early fraction to stop sooner
        return 1.0f;
    }
    ```
  </Tab>

  <Tab title="Performing Query">
    ```c joltc.h:1987-1995

    JPH_NarrowPhaseQuery_CastRay3(
        query,
        &origin,
        &direction,
        &settings,
        JPH_CollisionCollectorType_AllHit,  // Collect all hits
        OnRayHit,
        &ctx,
        NULL, NULL, NULL, NULL
    );
    
    printf("Found %d hits\n", ctx.hitCount);
    ```
  </Tab>
</Tabs>

## Shape Casting

Sweep a shape along a direction to find collisions.

### Basic Shape Cast

```c joltc.h:2035-2044
// Create shape to cast
JPH_SphereShape* sphere = JPH_SphereShape_Create(0.5f);

// Setup transform and direction
JPH_RMat4 startTransform;
JPH_RMat4_Identity(&startTransform);
startTransform.column3 = (JPH_RVec3){ 0, 1, 0 };

JPH_Vec3 direction = { 10.0f, 0.0f, 0.0f };

// Configure cast settings
JPH_ShapeCastSettings settings;
JPH_ShapeCastSettings_Init(&settings);
settings.base.activeEdgeMode = JPH_ActiveEdgeMode_CollideOnlyWithActive;
settings.backFaceModeTriangles = JPH_BackFaceMode_IgnoreBackFaces;
settings.useShrunkenShapeAndConvexRadius = true;
settings.returnDeepestPoint = false;

// Perform cast
JPH_RVec3 baseOffset = { 0, 0, 0 };
JPH_ShapeCastResult result;

bool hit = JPH_NarrowPhaseQuery_CastShape(
    query,
    (const JPH_Shape*)sphere,
    &startTransform,
    &direction,
    &settings,
    &baseOffset,
    MyShapeCastCallback,
    userData,
    NULL, NULL, NULL, NULL
);
```

## Practical Examples

### Ground Check

```c
JPH_RVec3 characterPos = { 0, 1, 0 };
JPH_Vec3 downDir = { 0, -1, 0 };

JPH_RayCastSettings settings;
settings.backFaceModeTriangles = JPH_BackFaceMode_IgnoreBackFaces;
settings.backFaceModeConvex = JPH_BackFaceMode_IgnoreBackFaces;
settings.treatConvexAsSolid = true;

JPH_RayCastResult hit;
bool onGround = JPH_NarrowPhaseQuery_CastRay(
    query,
    &characterPos,
    &downDir,
    &hit,
    NULL, NULL, NULL
);

if (onGround && hit.fraction < 0.1f) {
    // Character is on ground (within 0.1 units)
    JPH_RVec3 groundPoint;
    JPH_RRayCast_GetPointOnRay(&characterPos, &downDir, hit.fraction, &groundPoint);
}
```

### Line of Sight Check

```c
typedef struct {
    JPH_BodyID playerID;
    JPH_BodyID enemyID;
    bool hasLineOfSight;
} LineOfSightContext;

static float OnLineOfSightHit(void* context, const JPH_RayCastResult* result) {
    LineOfSightContext* ctx = (LineOfSightContext*)context;
    
    // If we hit the enemy, we have line of sight
    if (result->bodyID == ctx->enemyID) {
        ctx->hasLineOfSight = true;
        return result->fraction;  // Stop here
    }
    
    // If we hit something else, line of sight is blocked
    if (result->bodyID != ctx->playerID) {
        ctx->hasLineOfSight = false;
        return result->fraction;  // Stop here
    }
    
    // Skip player body, continue ray
    return 1.0f;
}

// Check line of sight
LineOfSightContext ctx = {
    .playerID = playerBodyID,
    .enemyID = enemyBodyID,
    .hasLineOfSight = false
};

JPH_RVec3 playerPos, enemyPos;
GetBodyPositions(&playerPos, &enemyPos);

JPH_Vec3 direction;
JPH_Vec3_Subtract((JPH_Vec3*)&enemyPos, (JPH_Vec3*)&playerPos, &direction);

JPH_NarrowPhaseQuery_CastRay3(
    query,
    &playerPos,
    &direction,
    NULL,
    JPH_CollisionCollectorType_ClosestHit,
    OnLineOfSightHit,
    &ctx,
    NULL, NULL, NULL, NULL
);

if (ctx.hasLineOfSight) {
    // Enemy is visible
}
```

### Sphere Overlap Test

```c
typedef struct {
    int bodyCount;
    JPH_BodyID bodies[32];
} OverlapContext;

static void OnBodyOverlap(void* context, const JPH_BodyID bodyID) {
    OverlapContext* ctx = (OverlapContext*)context;
    if (ctx->bodyCount < 32) {
        ctx->bodies[ctx->bodyCount++] = bodyID;
    }
}

// Find all bodies overlapping explosion radius
JPH_Vec3 explosionCenter = { 10, 1, 5 };
float explosionRadius = 5.0f;

OverlapContext ctx = { 0 };

JPH_BroadPhaseQuery_CollideSphere(
    broadPhase,
    &explosionCenter,
    explosionRadius,
    OnBodyOverlap,
    &ctx,
    NULL,
    NULL
);

// Apply explosion force to all overlapping bodies
for (int i = 0; i < ctx.bodyCount; i++) {
    ApplyExplosionForce(ctx.bodies[i], explosionCenter, explosionRadius);
}
```

## Best Practices

<AccordionGroup>
  <Accordion title="Choose the Right Query Type">
    - Use **broad phase** for fast, approximate tests (explosion radius, triggers)
    - Use **narrow phase** for accurate gameplay (shooting, ground check)
    - Use **shape casts** instead of raycasts for thick objects
    - Use **point queries** for simple inside/outside tests
  </Accordion>

  <Accordion title="Performance Optimization">
    - Use appropriate collector types (ClosestHit is faster than AllHit)
    - Return early from callbacks when possible
    - Use filters to skip unnecessary bodies
    - Prefer broad phase for large numbers of queries
    - Cache filters instead of recreating each query
  </Accordion>

  <Accordion title="Raycast Settings">
    - Set `treatConvexAsSolid = true` for solid objects
    - Use `IgnoreBackFaces` for one-way surfaces
    - Set `returnDeepestPoint = true` for penetration resolution
    - Use `useShrunkenShapeAndConvexRadius` for accurate shape casts
  </Accordion>

  <Accordion title="Filtering">
    - Use broadphase filter to skip entire layers
    - Use object layer filter for gameplay categories
    - Use body filter to skip specific bodies (player, etc.)
    - Use shape filter for fine-grained control
    - Filters are checked in order: broadphase → object layer → body → shape
  </Accordion>
</AccordionGroup>
